Financial Forecasting- Project
Objective:* Analyze historical financial statements to project future corporate revenue velocities and operational expense growth thresholds.
Tools Used:* Python (Time Series Analysis / ARIMA model), Microsoft Excel, Power BI.
Methodology:* Executed data cleaning, trend identification, and seasonality analysis on past transactional data.
Key Outcome:* Using predictive Time Series Modeling, I successfully generated a comprehensive 3-year financial outlook forecasting *Revenue, Expenses, Profit Before Tax (PBT), and Profit After Tax (PAT)** for the fiscal years **2027, 2028, and 2029*:
1. **Python Algorithmic Core:* Developed an auto-regressive Integrated Moving Average (ARIMA) script to process historical income statement line items and extrapolate raw mathematical projections.
2. **Excel Financial Modeling:* Ingested the python-generated projections into a structured corporate financial model to compute standard margins, tax provisions, PBT, and final PAT schedules.
3. **Power BI Executive Dashboard:* Designed an interactive forward-looking Nestlé India Financial Forecasting - Forecast Analysis (2027–2029)
